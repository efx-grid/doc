Content Wrap

The Content Wrap Extension is an extension that shows all text information in a cell, even if it is too small to show the text horizontally. The extension will automatically does a line break for content that longer than the available space in a cell.

Content Wrap extension cannot work together with row virtualization mechanism. Please make sure that rowVirtualization property is set to false in the configs for the extension to work correctly.

Content Wrap extension works by measuring row height once the content has been rendered, and then dynamically changing the row height. In contrast, row virtualization works by calculating number of rows to be displayed in the view, and not rendering any content outside of the view. Row virtualization requires row height to be constant in order to calculate the number of rows to be rendered, but the extension will keep changing row height after content is rendered. Hence, row virtualization cannot be enabled with the extension. So, be vary of some performance impact when using this extension.

Due to the performance concern, contentWrap: true has to specifically be set on the column that needs content to be wrapped.

Normal mode

efx-grid {
    height: 169px;
}

<efx-grid id="grid"></efx-grid>

await (await import("./theme-loader.js")).default; // This line is only required for demo purpose. It is not relevant for using EFX Grid in your application.

/* ---------------------------------- Note ----------------------------------
  DataGenerator, Formatters and extensions are exposed to global scope
  in the bundle file to make it easier to create live examples.
  Importing formatters and extensions is still required in your application. 
  Please see the document for further information.
---------------------------------------------------------------------------*/
var fields = ["companyName", "market", "CF_LAST", "CF_NETCHNG", "industry"];
var records = DataGenerator.generateRecords(fields, { seed: 0, numRows: 15 });
var configObj = {
    rowVirtualization: false,
    columns: [
        { title: "Company", field: fields[0] },
        { title: "Market", field: fields[1], width: 120 },
        { title: "Last", field: fields[2], width: 100 },
        { title: "Net. Chng", field: fields[3], width: 100 },
        { title: "Industry", field: fields[4], width: 120, contentWrap: true }
    ],
    staticDataRows: records,
    extensions: [
        new ContentWrap()
    ]
};

var grid = document.getElementById("grid");
grid.config = configObj;

var fields = ["companyName", "market", "CF_LAST", "CF_NETCHNG", "industry"];
var records = DataGenerator.generateRecords(fields, { seed: 0, numRows: 15 });
var configObj = {
    rowVirtualization: false,
    columns: [
        { title: "Company", field: fields[0] },
        { title: "Market", field: fields[1], width: 120 },
        { title: "Last", field: fields[2], width: 100 },
        { title: "Net. Chng", field: fields[3], width: 100 },

        new ContentWrap()

grid.config = configObj;
