Zoom

The Zoom Extension adds Grid's ability to zoom in or zoom out.

html hr {
    margin: 5px;
}
efx-grid {
    height: 208px;
}

<big>Ctrl + Mouse Scroll to zoom in/out.</big>
<hr>
<efx-grid id="grid"></efx-grid>

await (await import("./theme-loader.js")).default; // This line is only required for demo purpose. It is not relevant for using EFX Grid in your application.

/* ---------------------------------- Note ----------------------------------
  DataGenerator, Formatters and extensions are exposed to global scope
  in the bundle file to make it easier to create live examples.
  Importing formatters and extensions is still required in your application. 
  Please see the document for further information.
---------------------------------------------------------------------------*/
var zoomExt = new Zoom();
var fields = ["companyName", "market", "CF_LAST", "CF_NETCHNG", "industry"];
var records = DataGenerator.generateRecords(fields, { numRows: 10 });
var configObj = {
    columns: [
        { title: "Company", field: fields[0], width: 250, leftPinned: true },
        { title: "Market", field: fields[1], width: 160 },
        { title: "Last", field: fields[2], width: 160 },
        { title: "Net. Chng", field: fields[3], width: 160 },
        { title: "Industry", field: fields[4], width: 200, rightPinned: true }
    ],
    staticDataRows: records,
    extensions: [
        zoomExt
    ]
};

var grid = document.getElementById("grid");
grid.config = configObj;

Type Definitions

typedef

Options

Available options describing `zoom` object specified in grid's option
Type:
Object
Properties:
Name Type Attributes Default Description
zoomFactor number <optional>
 1 Default zoom level range from 0.6 to 2
step number <optional>
 0.1 Steps size of each zoom level
wheel boolean <optional>
 true If disabled, not allow wheel event to change zoom
maxFactor number <optional>
 2 Maximum zoom factor level
minFactor number <optional>
 0.5 Minimum zoom factor level
zoomChanged function <optional>
 null Event handler when zoom factor is changed through the APIs or mouse wheel.
