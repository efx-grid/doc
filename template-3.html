getColSpan(indexX, indexY)

Returns zero or negative value, if the cell is spanned over (i.e. occupied by the spanning cell).
Negative value is distance from the spanning cell (i.e. -1 means one cell from the spanning cell and -2 means two cells from the spanning cell)
Return one, if the cell has no span (normal cell).
Return value greater than one, if the cell is spanning (getting expanded)
