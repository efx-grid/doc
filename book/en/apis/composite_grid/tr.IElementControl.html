tr.IElementControl

All Novakit's UI controls is implementing this interface, which provides common element manipulations for ease of use. All standard HTML element events are allowed and can be used through tr.IElementControl#listen See all available event list here tr.IElementControl#event:blur

Events

event

blur

Fires the moment that the element loses focus
Example:
var popup = new tr.Popup();
popup.listen("blur", function(e) { console.log("A popup is out of focus"); });
event

change

Fires the moment when the value of the element is changed
event

click

Fires on a mouse click on the element
Example:
var grid = new tr.Grid();
grid.listen("click", function(e) { console.log("Grid is clicked"); });
event

contextmenu

Fires when a context menu is triggered
event

dblclick

Fires on a mouse double-click on the element
event

drag

Fires when an element is dragged
event

dragend

Fires at the end of a drag operation
event

dragenter

Fires when an element has been dragged to a valid drop target
event

dragleave

Fires when an element leaves a valid drop target
event

dragover

Fires when an element is being dragged over a valid drop target
event

dragstart

Fires at the start of a drag operation
event

drop

Fires when dragged element is being dropped
event

focus

Fires the moment when the element gets focus
event

formchange

Fires when a form changes
event

forminput

Fires when a form gets user input
event

input

Fires when an element gets user input
event

invalid

Fires when an element is invalid
event

keydown

Fires when a user is pressing a key
event

keypress

Fires when a user presses a key
event

keyup

Fires when a user releases a key
event

mousedown

Fires when a mouse button is pressed down on an element
event

mousemove

Fires when the mouse pointer moves over an element
event

mouseout

Fires when the mouse pointer moves out of an element
event

mouseover

Fires when the mouse pointer moves over an element
event

mouseup

Fires when a mouse button is released over an element
event

mousewheel

Fires when the mouse wheel is being rotated
event

reset

Fires when the Reset button in a form is clicked Not supported in HTML5
event

scroll

Fires when an element's scrollbar is being scrolled
event
