SortLogic()

SortLogic function is used for comparison between two values during sorting.
The function should return:
negative value to put val1 before val2,
positive value to put val2 before val1.
zero or falsy value to keep val1 and val2 unchanged.
Properties:
Name Type Attributes Description
val1 * Value from a row
val2 * Value from another row
order number <optional>
 1 indicates ascending order, and -1 indicates descending order
contextObject * <optional>
 Any given user context
Example:
var customComparer = function(a, b, order) {
  if(a + 10 < b) { return -order; }
  if(b + 10 < a) { return order; }
  return 0; // a and b are less than 10 difference apart
};

setDataFromObjectMap(obj)

Add, update, and remove data in batch by using JSON object. This is faster than repeatedly calling DataTable#setRowData.
Any existing row, that is not matched by the same row id provided in the argument, remains the same.
Any existing row with the same row id from the provided argument will be updated.
Row id with null value will cause a removal of the row with the same id.
Parameters:
obj
Object.<string, Object.<string, *>>
Object is 2 level nested object where 1st level key is row Id and 2nd level key is column Id
See:
  • DataTable#addRows
  • DataTable#setDataFrom2DArray
Example:
var data = {
  "rowId 1": {"column 1": "a", "column 2": 1, "column 3": 2},
  "rowId 2": {"column 1": "b", "column 2": 3, "column 3": 4},
  "rowId 4": {"column 1": "d", "group_criteria": 1} // Rows don't need to be in uniform structure
};
var dt = new DataTable();
dt.setDataFromObjectMap(data);
dt.log(); // See console for the results
dt.setDataFromObjectMap({"rowId 1": null}); // Remove the row with "rowId 1"
dt.setDataFromObjectMap({"rowId 2": {"column 1": "newValue"} }); // Update existing row
