Events

event

dataChanged

Trigger when data within the table has been changed
Properties:
Name Type Description
globalChange boolean Indicates a big change. User should expect all data has been change
type string Type of changes. Possible values are "inserted", "removed", "updated", and undefined
rid string Row Id
rowData Object.<string, *> Column values of the changed row in JSON object format
event

dataComposed

Trigger before DataCache#dataChanged. Perform any data update during the event will NOT cause more dataChanged events
