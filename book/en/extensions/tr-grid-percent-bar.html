Percent Bar

The Percent Bar Extension is an extension that provides formatter for rendering percent bar in column.

For contributions, check out the Percent Bar Extension repository.

  • HTML
  • JS
  • CSS
<emerald-grid id="grid">

</emerald-grid>
var fields = ["companyName", "market", "CF_LAST", "CF_NETCHNG", "industry"];
var records = tr.DataGenerator.generateRecords(fields, { numRows: 20 });
var configObj = {
  rowHighlight: true,
  rowSelection: true,
  columns: [
    { title: "Company", field: fields[0] },
    { title: "Market", field: fields[1], width: 120 },
    { title: "Last", field: fields[2], width: 100 },
    { title: "Net. Chng", field: fields[3], width: 150, percentBar: { alignment: "l" } },
    { title: "Industry", field: fields[4] }
  ],
  dataModel: {
    data: records
  },
  extensions: [
    new tr.PercentBarExtension()
  ]
};

var grid = document.getElementById("grid");
grid.config = configObj;
emerald-grid {
  height: 169px;
}

Type Definitions

typedef

ColumnOptions

The options can be specified by `percentBar` property in the grid's column definition
Type:
Object
Properties:
Name Type Attributes Default Description
alignment string <optional>
 By default percent bar will have the same alignment as the column
field string <optional>
 THIS_COLUMN Data field
textHidden boolean <optional>
 false If enabled, displays percent bar only
textWidth number | string <optional>
 If specified, percent bar text will have fixed width
useMovementColor boolean <optional>
 true If disabled, percent bar will not use up and down colors
padding number <optional>
 8 Extra space added to max width from autoTextSizing calculation
percentageLabel boolean <optional>
 true If disabled, the value will not convert into percent format
autoTextSizing boolean <optional>
 true If disabled, text space will be fixed
autoFitting boolean <optional>
 false If enabled, text space will be auto adjusted
but if disabled, text space will stay extended to the longest length
